752- (UIImage *)croppedImage:(UIImage *)image cropMode:(RSKImageCropMode)cropMode cropRect:(CGRect)cropRect rotationAngle:(CGFloat)rotationAngle zoomScale:(CGFloat)zoomScale maskPath:(UIBezierPath *)maskPath applyMaskToCroppedImage:(BOOL)applyMaskToCroppedImage
753{
754 // Step 1: check and correct the crop rect.
755 CGSize imageSize = image.size;
756 CGFloat x = CGRectGetMinX(cropRect);
757 CGFloat y = CGRectGetMinY(cropRect);
758 CGFloat width = CGRectGetWidth(cropRect);
759 CGFloat height = CGRectGetHeight(cropRect);
760
761 UIImageOrientation imageOrientation = image.imageOrientation;
762 if (imageOrientation == UIImageOrientationRight || imageOrientation == UIImageOrientationRightMirrored) {
763 cropRect.origin.x = y;
764 cropRect.origin.y = round(imageSize.width - CGRectGetWidth(cropRect) - x);
765 cropRect.size.width = height;
766 cropRect.size.height = width;
767 } else if (imageOrientation == UIImageOrientationLeft || imageOrientation == UIImageOrientationLeftMirrored) {
768 cropRect.origin.x = round(imageSize.height - CGRectGetHeight(cropRect) - y);
769 cropRect.origin.y = x;
770 cropRect.size.width = height;
771 cropRect.size.height = width;
772 } else if (imageOrientation == UIImageOrientationDown || imageOrientation == UIImageOrientationDownMirrored) {
773 cropRect.origin.x = round(imageSize.width - CGRectGetWidth(cropRect) - x);
774 cropRect.origin.y = round(imageSize.height - CGRectGetHeight(cropRect) - y);
775 }
776
777 CGFloat imageScale = image.scale;
778 cropRect = CGRectApplyAffineTransform(cropRect, CGAffineTransformMakeScale(imageScale, imageScale));
779
780 // Step 2: create an image using the data contained within the specified rect.
781 CGImageRef croppedCGImage = CGImageCreateWithImageInRect(image.CGImage, cropRect);
782 UIImage *croppedImage = [UIImage imageWithCGImage:croppedCGImage scale:imageScale orientation:imageOrientation];
783 CGImageRelease(croppedCGImage);
784
785 // Step 3: fix orientation of the cropped image.
786 croppedImage = [croppedImage fixOrientation];
787 imageOrientation = croppedImage.imageOrientation;
788
789 // Step 4: If current mode is `RSKImageCropModeSquare` and the image is not rotated
790 // or mask should not be applied to the image after cropping and the image is not rotated,
791 // we can return the cropped image immediately.
792 // Otherwise, we must further process the image.
793 if ((cropMode == RSKImageCropModeSquare || !applyMaskToCroppedImage) && rotationAngle == 0.0) {
794 // Step 5: return the cropped image immediately.
795// return croppedImage;
796 return [self cutCenterImage:croppedImage];
797 } else {
798 // Step 5: create a new context.
799 CGSize maskSize = CGRectIntegral(maskPath.bounds).size;
800 CGSize contextSize = CGSizeMake(ceil(maskSize.width / zoomScale)__tg_ceil((__typeof__(__tg_promote((maskSize.width / zoomScale
))))(maskSize.width / zoomScale)),
801 ceil(maskSize.height / zoomScale)__tg_ceil((__typeof__(__tg_promote((maskSize.height / zoomScale
))))(maskSize.height / zoomScale)));
802 UIGraphicsBeginImageContextWithOptions(contextSize, NO__objc_no, imageScale);
803
804 // Step 6: apply the mask if needed.
805 if (applyMaskToCroppedImage) {
806 // 6a: scale the mask to the size of the crop rect.
807 UIBezierPath *maskPathCopy = [maskPath copy];
808 CGFloat scale = 1 / zoomScale;
809 [maskPathCopy applyTransform:CGAffineTransformMakeScale(scale, scale)];
810
811 // 6b: move the mask to the top-left.
812 CGPoint translation = CGPointMake(-CGRectGetMinX(maskPathCopy.bounds),
813 -CGRectGetMinY(maskPathCopy.bounds));
814 [maskPathCopy applyTransform:CGAffineTransformMakeTranslation(translation.x, translation.y)];
815
816 // 6c: apply the mask.
817 [maskPathCopy addClip];
818 }
819
820 // Step 7: rotate the cropped image if needed.
821 if (rotationAngle != 0) {
822 croppedImage = [croppedImage rotateByAngle:rotationAngle];
823 }
824
825 // Step 8: draw the cropped image.
826 CGPoint point = CGPointMake(round((contextSize.width - croppedImage.size.width) * 0.5f),
827 round((contextSize.height - croppedImage.size.height) * 0.5f));
828 [croppedImage drawAtPoint:point];
829
830 // Step 9: get the cropped image affter processing from the context.
831 croppedImage = UIGraphicsGetImageFromCurrentImageContext();
832
833 // Step 10: remove the context.
834 UIGraphicsEndImageContext();
835
836 croppedImage = [UIImage imageWithCGImage:croppedImage.CGImage scale:imageScale orientation:imageOrientation];
837
838 // Step 11: return the cropped image affter processing.
839 return croppedImage;
840 }
841}

843#pragma mark 根据size截取图片中间矩形区域的图片 这里的size是正方形
844-(UIImage *)cutCenterImage:(UIImage *)image {
845 CGSize imageSize = image.size;
846 if (imageSize.width == imageSize.height)
847 return image;
848
849 CGRect rect;
850 CGSize contentSize;
851 //初始化图片的显示位置，以及UIGraphicsBeginImageContext的显示范围
852 if (imageSize.height < imageSize.width) {
853 contentSize = CGSizeMake(imageSize.height, imageSize.height);
854// float leftMargin = (imageSize.height - imageSize.width) * 0.5;
855// rect = CGRectMake(leftMargin, 0, imageSize.width, imageSize.height);
856 rect = CGRectMake(0, 0, imageSize.height, imageSize.height);
857 }else{
858 contentSize = CGSizeMake(imageSize.width, imageSize.width);
859// float topMargin = (imageSize.width - imageSize.height) * 0.5;
860// rect = CGRectMake(0, topMargin, imageSize.width, imageSize.height);
861 rect = CGRectMake(0, 0, imageSize.width, imageSize.width);
862 }
863 CGImageRef imageRef = image.CGImage;
864 CGImageRef imageRefRect = CGImageCreateWithImageInRect(imageRef, rect);
865 UIImage *tmp = [[UIImage alloc] initWithCGImage:imageRefRect];
866 CGImageRelease(imageRefRect);
867
868 // 新建一个UIGraphicsBeginImageContext，设置好背景颜色，把图片放到相应位置，截图，生成新的方形图片
869 UIGraphicsBeginImageContext(contentSize);
870 CGContextRef context = UIGraphicsGetCurrentContext();
871 CGContextSetFillColorWithColor(context, [[UIColor whiteColor] CGColor]);
872 UIRectFill(CGRectMake(0, 0, contentSize.width, contentSize.height));
873 [tmp drawInRect:rect];
874 tmp = UIGraphicsGetImageFromCurrentImageContext();
875 UIGraphicsEndImageContext();
876 return tmp;
877}
